①課題名　お買い物メモ帳アプリ

②課題内容（どんな作品か）

今日の献立（朝・夜）を選べる買い物メモアプリ
朝ごはんはたまごかけごはん・みそ汁とトースト・目玉焼きから選択
夜ごはんはタイの塩昆布和え・サーモンとごまの和え物・とろたくから選択
献立に必要な食材が表示される
買い物メモに手動でアイテムを追加・チェック・削除できる
localStorageでページをリロードしてもデータが残る

③アプリのデプロイURL
https://makifavor-ai.github.io/20260527_memoapri/

④アプリのログイン用IDまたはPassword（ある場合）
なし

⑤工夫した点・こだわった点

献立を選んだら必要な食材が自動で表示されるようにした
localStorageで買い物リストがリロード後も消えないようにした
選び直しボタンで献立を変更できるようにした

⑥難しかった点・次回トライしたいこと（又は機能）

localStorageへの配列の保存と読み込み
次回は献立を選んだら食材が買い物リストに自動追加される機能を実装したい

⑦フリー項目

感想：localStorageを使うことでページをまたいでデータを保持できる仕組みが理解できた
